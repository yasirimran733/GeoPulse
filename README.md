GeoPulse — Smart Geo-Personalized Dashboard

GeoPulse is a modern React-based web application that automatically detects a user's location via IP and dynamically personalizes content such as weather, news, and regional data in real time.

This project demonstrates how real-world applications use geolocation to enhance user experience through intelligent content delivery.

 Features
 Automatic location detection using IP
 Real-time weather updates based on user city
 Localized news feed by country
 Currency awareness and conversion
 Dynamic UI based on user timezone
 Optimized API handling with clean architecture

 Tech Stack

React (Hooks + Context API)
Axios / Fetch for API calls
Modular API architecture
Custom Hooks for reusable logic
 APIs Used
IP Geolocation API (for user location detection)
Weather API (for real-time weather data)
News API (for localized headlines)

Scalable frontend architecture
